# 地物-光谱一致性检验报告（后半部分：地物 13–23）

> 说明：本报告对 `output_figures_reflectance/comparison` 目录下编号 13–23 的对比图进行人工目视检验，判断右侧照片所示地物与左侧反射光谱曲线是否大致对应。

## 总体结论

- **大致对应**：13、14、15、16、18、19、20、21、22、23（10/11）
- **基本可接受但需注意异常**：17（1/11）
- **明显不匹配**：0（0/11）

后半部分整体对应情况较好，但 SWIR（1400–2500 nm）区域的水汽吸收带/仪器噪声尖峰问题依旧普遍存在，需在后续数据处理中重点关注。

## 关键发现

1. **植被及类植被地物（18、19、21、22）**：均呈现明显的绿峰、红边、NIR 平台及水吸收谷，光谱与照片对应度高。
2. **干枯植被（20）**：反射率随波长单调上升，无 NIR 平台，符合枯叶/干枯有机质特征。
3. **人工地物（13、14、16、17）**：光谱形态差异明显，其中深色橡胶（14）反射率最低，红褐色橡胶地面（16）中等，石台（13）与蓝色大棚（17）的 SWIR 段及绝对反射率需进一步复核。
4. **水体（15）**：可见光蓝绿峰与近红外低反射特征清晰，验证了“水体在 NIR/SWIR 强吸收”的典型规律。
5. **无机地物（23）**：石头/裸土单调上升光谱与照片一致。

## 建议

- 对 **13、17** 号的可见光–NIR 绝对反射率以及 **13、14、15、16、17、18、19、20、22、23** 号在 1400 nm、1900 nm、2400 nm 附近的 SWIR 尖峰进行去噪、大气校正或测量条件复核。
- 若后续需要定量分析，建议将 SWIR 水汽吸收带（约 1350–1450 nm、1800–2000 nm）标记为不可靠波段。
- 对于 **17** 号蓝色大棚，可补充一次正对薄膜、避免背景散射的测量，以验证蓝波段反射率是否偏低。
